TDAT:  protected void center(Component cmp) {
  Dimension screenDim = Toolkit.getDefaultToolkit().getScreenSize();
  Rectangle frameDim = cmp.getBounds();
  cmp.setLocation(
   (screenDim.width - frameDim.width) / 2,
   (screenDim.height - frameDim.height) / 2);
 }

COM:centers a given Component on the screen
